# My-Learning-App

Windows

1. Download and install Node.js 8.x from https://nodejs.org/en/.
2. Download and install Git from https://git-scm.com/downloads.
3. Download and install Yarn from https://yarnpkg.com/en/docs/install.
4. Launch cmd.exe as administrator and run command ``npm install``
5. To run, run command ``npm run start``
6. Open browser and go to http://localhost:4200
